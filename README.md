# TP 01 - Landing page Novis



### Expliquez ce qu’est le Layout Shift.
Le layout shift est un mouvement inattendu d'un élément lors du chargement ou de l'utilisation de la page.

---

### Expliquez à quoi sert aria-label.
C'est un text qui n'est pas visible directement, mais est utile dans un cas d\'accessiblité, pour les lecteurs d'écran, dans le cas d'un bouton avec seulement une icon...

---

### On utilise souvent loading="lazy” expliquez son utilité.
Le lazy loading permet de retarder le chargement de certains éléments pour améliorer les performances et la vitesse de chargement de la page web.
